# 🏙️ CityVoice

### SDG-Based Civic Issue Reporting & Automation Platform

CityVoice is a lightweight civic issue reporting platform built to simplify how citizens report public infrastructure and environmental issues. Citizens can submit complaints through a web interface, and the entire reporting process is automated using n8n.

The project demonstrates workflow automation, webhook integration, Google Sheets, Gmail automation, and frontend development while aligning with the United Nations Sustainable Development Goals (SDGs).

---

## 🌍 Sustainable Development Goals (SDGs)

This project contributes to:

- 🏙️ SDG 11 – Sustainable Cities and Communities
- ⚖️ SDG 16 – Peace, Justice and Strong Institutions
- 🤝 SDG 17 – Partnerships for the Goals

---

## ✨ Features

- Submit civic complaints through a simple web interface
- Automatic Issue ID generation
- Complaint storage in Google Sheets
- Citizen confirmation email
- Department notification email
- Workflow automation using n8n
- Easy deployment and customization
